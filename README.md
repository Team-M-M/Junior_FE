### 📌 commit 규칙

> 1일 1 커밋을 목표로 활동 합니다.

- commit message: [Project name] [Board number] 간략한 내용 한 줄 띄고 세부 내용 / pr까지 적용
- 터미널에서 작성법:

```bash
git commit -m "[Funtional] [#12-2] using iterator

in js using Array prototype map in custom filter"
```
